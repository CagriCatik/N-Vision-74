# N-Vision-74 Blueprints And Wallpapers

This repository contains high–resolution blueprint/wallpaper images of the Hyundai N Vision 74 concept.  
All files are provided in multiple resolutions and background variants to support:

- Desktop and mobile wallpapers  
- CAD background images for sketching and modeling  
- Presentation and documentation visuals
