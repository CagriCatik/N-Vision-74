# A1 mini PLA Profile Notes

## Use case

Use this profile for fast PLA prototypes, tolerance coupons, detail parts, small brackets, mirrors, lights, badges, alignment pins, and small wheel/hub test parts.

## CAD assumptions

| Parameter | Starting value |
|---|---:|
| Maximum safe part envelope | below 180 x 180 x 180 mm |
| Wall thickness | 1.6-2.0 mm |
| Assembly clearance | 0.25-0.40 mm |
| Pin/hole clearance | 0.20-0.35 mm |
| Minimum detail thickness | 0.8-1.0 mm |

## Slicer notes

```text
Printer: Bambu Lab A1 mini
Material: PLA
Layer height: 0.16-0.20 mm
Walls: 3
Infill: 10-30%, depending on part
Supports: minimize
Brim: optional for tall or narrow parts
```

## Design warnings

- Do not use the A1 mini as the main printer for large ABS parts.
- Design extra split lines only if a body section must fit the A1 mini.
- Use it mainly to reduce iteration time before committing to large P1S prints.
