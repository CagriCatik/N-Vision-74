Quick Start Guide¶

Get your N-Vision-74 project up and running with this quick start guide.

Before You Begin¶

Required Tools¶

  • Allen Hex Keys (2mm, 2.5mm, 3mm, 4mm, 5mm)
  • Screwdrivers (Phillips #0, #1, #2)
  • Socket Wrench Set (10mm, 12mm, 14mm)
  • Calipers or Ruler
  • Small hammer or mallet
  • Soldering iron (for electronics)

Recommended Tools¶

  • Digital multimeter for testing
  • Loctite removable thread-lock
  • Small files for edge finishing
  • 3D printer (for custom parts)
  • Computer with SolidWorks/Fusion360

Safety Equipment¶

  • Safety glasses
  • Work gloves
  • Dust mask (for sanding/grinding)
  • Proper ventilation for soldering

Assembly Checklist¶

Phase 1: Chassis Foundation¶

  • Gather all chassis parts (Items 00-06, 04, 05, 06)
  • Verify all dimensions against CAD
  • Clean and deburr aluminum edges
  • Assemble main plate with braces
  • Install corner brackets
  • Verify squareness and flatness

Estimated Time: 1-2 hours

Phase 2: Suspension Installation¶

  • Install front tower assembly
  • Mount front suspension arms
  • Install rear tower assembly
  • Mount rear suspension arms
  • Install shock absorbers (4 total)
  • Test arm movement range

Estimated Time: 2-3 hours

Phase 3: Drivetrain Assembly¶

  • Assemble differential unit
  • Install center differential
  • Mount motor supports
  • Attach transmission components
  • Install drivetrain braces
  • Check for smooth rotation

Estimated Time: 1.5-2 hours

Phase 4: Electronics Setup¶

  • Mount battery trays
  • Install ESC in enclosure
  • Mount servo with brackets
  • Connect motor to ESC
  • Mount fan shroud for cooling
  • Test all electrical connections

Estimated Time: 1-2 hours

Phase 5: Steering Assembly¶

  • Install steering tower
  • Mount servo arm
  • Connect steering linkages
  • Adjust tie rod lengths
  • Center steering neutral position
  • Test full steering range

Estimated Time: 1 hour

Phase 6: Body & Trim¶

  • Position main shell on chassis
  • Install front bumper
  • Install rear bumper
  • Add mirrors and accessories
  • Apply decals and graphics
  • Final visual inspection

Estimated Time: 1-2 hours

First Run Preparation¶

Pre-Run Checklist¶

  • All fasteners tight (check with hex key)
  • Suspension arms move freely
  • Steering responds smoothly
  • Motor spins freely without load
  • Battery fully charged
  • ESC programmed correctly
  • Transmitter paired with receiver

Initial Setup Steps¶

  1. Calibrate Servo Center 

    1. Power on ESC with throttle stick centered
2. Steering should be centered
3. Adjust trim if needed

  2. Test Motor Direction 

    1. Connect battery
2. Slowly increase throttle
3. Motor should spin smoothly
4. Reverse throttle to stop

  3. Brake Testing 

    1. Apply gentle brake
2. Vehicle should stop smoothly
3. Check wheel alignment after stop

  4. Steering Test 

    1. Center steering
2. Turn full left - measure angle
3. Turn full right - measure angle
4. Angles should be equal

Maintenance Schedule¶

After First Run¶

  • Inspect all fasteners
  • Check for cracks or damage
  • Verify wheel alignment
  • Clean out dust and debris

Weekly (Regular Use)¶

  • Clean exterior body
  • Check tire wear and pressure
  • Inspect suspension arms
  • Test steering response

Monthly¶

  • Deep clean electronics
  • Lubricate suspension joints
  • Check differential fluid level
  • Test battery voltage
  • Inspect for cracks/damage

Before Events/Competition¶

  • Full system inspection
  • Replace worn tires if needed
  • Fresh bearing grease
  • Test all electronics
  • Verify fastener torque specs

Troubleshooting¶

Vehicle Won't Move¶

  1. Check battery voltage (should be 7.2V+)
  2. Verify motor connections
  3. Test ESC power light
  4. Check for mechanical blockage

Steering Not Responding¶

  1. Check servo power connection
  2. Verify transmitter is on
  3. Check servo connector seating
  4. Test servo manually

Suspension Feels Stiff¶

  1. Check shock oil level
  2. Look for dirt in joints
  3. Verify fastener tightness
  4. Test arm movement manually

Overheating Issues¶

  1. Check fan shroud placement
  2. Verify motor air intake is clear
  3. Reduce throttle input gradually
  4. Allow cooldown between runs

Performance Tuning¶

Suspension Geometry¶

  • Adjust shock length for ride height
  • Modify arm angles for handling
  • Test on different terrains

Motor Configuration¶

  • 4000 KV is good for general use
  • Higher KV = faster, more current draw
  • Gear ratio affects speed/torque balance

Steering Sensitivity¶

  • Servo center determines neutral
  • Endpoint adjustment changes throw
  • Test in empty area first
