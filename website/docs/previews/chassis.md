# Chassis Components

Interactive 3D preview of the N-Vision-74 chassis assembly.

## Overview

The chassis is the structural backbone of the vehicle, providing mounting points for all major systems including suspension, drivetrain, and electronics.

## Main Chassis Plate

The primary structural member that integrates all subsystems.

**Part Number:** 04_Chassis_Main_Plate.SLDPRT  
**Material:** Aluminum 6061 (3mm)  
**Weight:** ~200g  
**Dimensions:** 260mm L × 140mm W × 3mm H

### Features
- CNC-machined aluminum construction
- Precision-drilled mounting holes
- Integrated cable routing channels
- Reinforced corner sections
- Lightweight yet rigid design
- Anti-corrosion anodized finish

## Assembly Process

### Step 1: Prepare Main Plate
- Inspect for damage
- Debur all edges
- Clean with compressed air
- Place on flat work surface

### Step 2: Install Front Module
1. Position front module at front of main plate
2. Align all mounting holes
3. Insert M4 socket head screws
4. Tighten in X-pattern (cross pattern)
5. Torque to 3-4 Nm

### Step 3: Install Rear Module
1. Position rear module at rear of main plate
2. Align with reference marks
3. Insert M4 screws with lock washers
4. Tighten in X-pattern
5. Torque to 3-4 Nm

## Fastener Specifications

### Socket Head Cap Screws

| Size | Location | Quantity | Torque |
|------|----------|----------|--------|
| M3 | Bracing | 8 | 1.5-2.0 Nm |
| M4 | Module mount | 12 | 3-4 Nm |
| M5 | Tower mount | 8 | 5-6 Nm |

### Lock Washers & Nuts

- Use lock washers under all screw heads
- Use flanged nuts for open holes
- Flanged nut size: M4 (minimum 10 pieces)

## Modification Guide

### Common Upgrades

1. **Carbon Fiber Bracing**
   - Reduces weight by 20-30%
   - Improves rigidity
   - Cost: $30-50 additional

2. **Aluminum Bracing Upgrade**
   - More durable than carbon fiber
   - Better for harsh conditions
   - Cost: $15-25 additional

3. **Custom Electronics Mount**
   - Move ESC/receiver location
   - Requires custom drilling
   - Improves weight distribution

4. **Ballast Mount**
   - Add adjustable weight system
   - Fine-tune balance and handling
   - Cost: $20-40

### Performance Considerations

| Modification | Effect | Difficulty |
|--------------|--------|-----------|
| Carbon Fiber Bracing | +15% stiffness, -20% weight | Medium |
| Aluminum Upgrade | +30% stiffness | Low |
| Electronics Relocation | Better balance | Medium |
| Ballast System | Tunable handling | High |

## Maintenance Schedule

### After Each Run
- [ ] Inspect for cracks
- [ ] Check fastener tightness
- [ ] Clean mud and debris
- [ ] Dry thoroughly

### Weekly (Regular Use)
- [ ] Detailed fastener inspection
- [ ] Check alignment (straight line test)
- [ ] Inspect welded/brazed joints
- [ ] Test flexing by hand

### Monthly
- [ ] Re-torque all fasteners
- [ ] Deep clean
- [ ] Inspect for fatigue cracks
- [ ] Check for corrosion

## Storage & Care

### Storage Best Practices
- Store in dry environment
- Avoid direct sunlight (UV damage)
- Protect from moisture
- Keep fasteners in sealed container

### Corrosion Prevention
- Use stainless steel fasteners
- Apply anodized coating if damaged
- Store with desiccant packets
- Check for oxidation regularly
