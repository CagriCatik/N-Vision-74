# N-Vision-74 Documentation

Welcome to the N-Vision-74 project documentation. This comprehensive guide covers everything you need to know about building and customizing your 1/10 scale vehicle.

## Project Overview

N-Vision-74 is a sophisticated 1/10 scale vehicle platform featuring:

- **Modular Assembly Structure** - Easy-to-assemble components with clear part organization
- **Detailed CAD Models** - Professional SolidWorks files for each component
- **3D Component Previews** - Interactive visualization of all major assemblies
- **Complete Documentation** - From BOM to assembly guides and technical specs

## Key Features

### Interactive 3D Viewer
All component pages include interactive 3D previews. Use your mouse to rotate, zoom, and inspect the models.

### Assembly-Driven Structure
The vehicle is organized using an assembly-driven SolidWorks structure, making it easy to understand and modify individual components.

### Complete Part Naming
All parts follow a consistent naming convention with revision tracking.
