# Quick Start Guide

Get your N-Vision-74 project up and running with this quick start guide.

## Before You Begin

### Required Tools

- Allen Hex Keys (2mm, 2.5mm, 3mm, 4mm, 5mm)
- Screwdrivers (Phillips #0, #1, #2)
- Socket Wrench Set (10mm, 12mm, 14mm)
- Calipers or Ruler
- Small hammer or mallet
- Soldering iron (for electronics)

### Recommended Tools

- Digital multimeter for testing
- Loctite removable thread-lock
- Small files for edge finishing
- 3D printer (for custom parts)
- Computer with SolidWorks/Fusion360

### Safety Equipment

- Safety glasses
- Work gloves
- Dust mask (for sanding/grinding)
- Proper ventilation for soldering

## First Run Preparation

### Pre-Run Checklist

- [ ] All fasteners tight (check with hex key)
- [ ] Suspension arms move freely
- [ ] Steering responds smoothly
- [ ] Motor spins freely without load
- [ ] Battery fully charged
- [ ] ESC programmed correctly
- [ ] Transmitter paired with receiver

### Initial Setup Steps

1. **Calibrate Servo Center**
   ```
   1. Power on ESC with throttle stick centered
   2. Steering should be centered
   3. Adjust trim if needed
   ```

2. **Test Motor Direction**
   ```
   1. Connect battery
   2. Slowly increase throttle
   3. Motor should spin smoothly
   4. Reverse throttle to stop
   ```

3. **Brake Testing**
   ```
   1. Apply gentle brake
   2. Vehicle should stop smoothly
   3. Check wheel alignment after stop
   ```

4. **Steering Test**
   ```
   1. Center steering
   2. Turn full left - measure angle
   3. Turn full right - measure angle
   4. Angles should be equal
   ```

## Troubleshooting

### Vehicle Won't Move
1. Check battery voltage (should be 7.2V+)
2. Verify motor connections
3. Test ESC power light
4. Check for mechanical blockage

### Steering Not Responding
1. Check servo power connection
2. Verify transmitter is on
3. Check servo connector seating
4. Test servo manually

### Suspension Feels Stiff
1. Check shock oil level
2. Look for dirt in joints
3. Verify fastener tightness
4. Test arm movement manually

### Overheating Issues
1. Check fan shroud placement
2. Verify motor air intake is clear
3. Reduce throttle input gradually
4. Allow cooldown between runs

## Performance Tuning

### Suspension Geometry
- Adjust shock length for ride height
- Modify arm angles for handling
- Test on different terrains

### Motor Configuration
- 4000 KV is good for general use
- Higher KV = faster, more current draw
- Gear ratio affects speed/torque balance

### Steering Sensitivity
- Servo center determines neutral
- Endpoint adjustment changes throw
- Test in empty area first
