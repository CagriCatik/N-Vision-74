# Detailed Assembly Guide

Step-by-step instructions for assembling your N-Vision-74 vehicle.

## Phase 1: Chassis Assembly

### Step 1.1: Prepare Main Plate

1. Retrieve main chassis plate (Part 04)
2. Inspect for damage or manufacturing defects
3. Debur all edges with a file
4. Clean with compressed air
5. Place on flat, stable work surface

### Step 1.2: Install Front Module

1. Position front module (Part 05) at front of main plate
2. Align mounting holes precisely
3. Insert M4 socket head screws into corner brackets
4. Tighten sequentially in X-pattern to prevent warping
5. **Torque spec:** M4 = 3-4 Nm
6. Verify alignment with calipers

### Step 1.3: Install Rear Module

1. Position rear module (Part 06) at rear of main plate
2. Align with reference marks on main plate
3. Insert M4 screws with lock washers
4. Tighten in X-pattern
5. **Torque spec:** M4 = 3-4 Nm
6. Ensure perpendicular alignment

### Step 1.4: Add Cross Braces

1. Position carbon fiber brace on top surface
2. Mark mounting hole locations
3. Drill if necessary (use pilot holes first)
4. Install with M3 screws and lock washers
5. **Torque spec:** M3 = 1.5-2 Nm
6. Check for symmetry

### Step 1.5: Final Chassis Check

- [ ] All fasteners tight
- [ ] No visible gaps at joints
- [ ] Chassis sits flat on table
- [ ] No warping or twisting

**Completion Time:** 45-60 minutes

## Phase 2: Suspension Assembly

### Front Suspension

#### Step 2.1: Install Front Tower

1. Position front tower assembly at front of chassis
2. Align dowel pins with mounting holes
3. Insert M5 socket head screws (4 total)
4. Tighten gradually to proper torque
5. **Torque spec:** M5 = 5-6 Nm

#### Step 2.2: Mount Front Suspension Arms

1. Insert top arm pivots into tower mounts
2. Secure with M4 socket head screws
3. Install bottom arm pivots
4. Add lock washers under all screw heads
5. **Torque spec:** M4 = 3-4 Nm
6. Verify smooth pivoting action

#### Step 2.3: Install Shock Absorbers

1. Position shock absorber (front-left)
2. Mount upper ball joint to tower arm mount
3. Mount lower ball joint to chassis bracket
4. Repeat for front-right shock
5. Test compression and rebound smoothly

**Front Suspension Time:** 1.5-2 hours

### Rear Suspension

#### Step 2.4: Install Rear Tower

1. Position rear tower assembly at rear of chassis
2. Align with mounting plate
3. Insert M5 socket head screws (4 total)
4. Tighten to proper torque
5. **Torque spec:** M5 = 5-6 Nm

#### Step 2.5: Mount Rear Suspension Arms

1. Insert arm pivots into tower
2. Secure with M4 socket head screws
3. Add lock washers
4. **Torque spec:** M4 = 3-4 Nm
5. Verify movement range

#### Step 2.6: Install Rear Shocks

1. Position shock absorber (rear-left)
2. Mount to arm and chassis brackets
3. Repeat for rear-right shock
4. Test full suspension movement
5. Check for rubbing or interference

**Rear Suspension Time:** 1.5-2 hours

## Phase 3: Drivetrain Assembly

### Step 3.1: Prepare Differential

1. Remove differential from package
2. Inspect all gears and bearings
3. Clean with light oil if stored
4. Check for smooth rotation

### Step 3.2: Install Motor

1. Position brushless motor on motor mounts
2. Align with differential shaft
3. Secure with M3 socket head screws (4 total)
4. **Torque spec:** M3 = 1.5-2 Nm
5. Check alignment - no rubbing

### Step 3.3: Connect Transmission

1. Align motor pinion with differential ring gear
2. Install motor-to-diff connector
3. Test rotation - should be smooth
4. Check gear mesh - no binding

### Step 3.4: Install Differential Housing

1. Position differential unit in chassis center
2. Align mounting holes with chassis
3. Install mounting brackets
4. Secure with M4 socket head screws
5. **Torque spec:** M4 = 3-4 Nm

### Step 3.5: Add Support Braces

1. Install center brace holders
2. Mount braces to prevent deflection
3. Verify no interference with moving parts

**Drivetrain Assembly Time:** 1.5-2 hours

## Phase 4: Electronics Installation

### Step 4.1: Prepare Electronics Enclosure

1. Mount battery trays (L/R) on chassis sides
2. Secure with M3 screws
3. Install ESC in weatherproof enclosure
4. Mount enclosure on top of chassis

### Step 4.2: Motor Connection

1. Connect motor leads to ESC:
   - Red = Positive
   - Black = Negative
   - Connection type depends on ESC
2. Solder connections with rosin core solder
3. Apply heatshrink tubing over joints
4. Test continuity with multimeter

### Step 4.3: Servo Installation

1. Mount servo in servo bracket
2. Secure with servo clips/screws
3. Connect servo cable to ESC receiver channel 1
4. Run cable along chassis frame

### Step 4.4: Battery Mounting

1. Place LiPo battery in battery trays
2. Secure with velcro or straps
3. Ensure battery is centered for weight balance
4. Connect to ESC battery connector

### Step 4.5: Final Electronics Check

- [ ] All connections seated properly
- [ ] No exposed wires or connectors
- [ ] Servo moves when powered
- [ ] Motor responds to throttle
- [ ] Cooling fan works

**Electronics Installation Time:** 1-1.5 hours

## Phase 5: Steering Assembly

### Step 5.1: Mount Servo Arm

1. Install servo arm on servo shaft
2. Tighten servo arm screw securely
3. Servo should be centered position
4. Mark neutral position with tape

### Step 5.2: Connect Steering Linkage

1. Connect servo arm to steering tower
2. Use ball end links for pivot points
3. Adjust length for neutral steering
4. Test full steering range

### Step 5.3: Tie Rod Setup

1. Mount tie rods front and rear
2. Adjust lengths for toe-in alignment
3. Set to manufacturer specifications
4. Tighten lock nuts carefully

### Step 5.4: Test Steering Response

1. Center steering with transmitter
2. Verify vehicle steers straight
3. Test full left turn
4. Test full right turn
5. Check for binding or resistance

**Steering Assembly Time:** 45-60 minutes

## Phase 6: Body and Trim Installation

### Step 6.1: Install Main Body Shell

1. Position body shell on chassis
2. Align with mounting posts
3. Secure with body clips or fasteners
4. Ensure no interference with moving parts

### Step 6.2: Attach Bumpers

1. Position front bumper (Part 02)
2. Secure with provided fasteners
3. Install rear bumper (Part 03)
4. Check alignment and clearance

### Step 6.3: Add Accessories

1. Install mirrors (Parts 09, 10)
2. Mount light housings (Part 11)
3. Install interior tub (Part 12)
4. Add any custom decals or graphics

### Step 6.4: Final Inspection

- [ ] All body parts secure
- [ ] No sharp edges exposed
- [ ] Wheels turn freely
- [ ] Vehicle is symmetrical
- [ ] All lights/mirrors functional

**Body Installation Time:** 1-1.5 hours

## Assembly Tips & Best Practices

### General Guidelines
- ✓ Work in clean, organized workspace
- ✓ Organize screws by size/function
- ✓ Take photos during disassembly for reference
- ✓ Use calibrated torque wrench for precision
- ✓ Never force parts - check alignment first
- ✓ Allow solder joints to cool before moving

### Common Mistakes to Avoid
- ✗ Over-tightening fasteners (causes stripping)
- ✗ Skipping lock washers (fasteners loosen)
- ✗ Incorrect servo orientation (reversed steering)
- ✗ Poor motor-diff alignment (binding/noise)
- ✗ Neglecting threadlock (fasteners fall out)

## Quality Control

After assembly completion:

1. **Visual Inspection** - Check all joints, no gaps
2. **Movement Test** - Suspension, steering, wheels
3. **Electrical Test** - Power, motor, servo responses
4. **Torque Audit** - Re-check all fasteners
5. **Run-through** - Slow, careful test drive
